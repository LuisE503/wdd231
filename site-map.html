Space Snapshot

This name was chosen because it reflects the dynamic and awe-inspiring nature of space, showcased through NASA's free API images.

Site Purpose

The site will serve as an informational hub about NASA, offering three main pages: a Home Page featuring a quick introduction to NASA and space topics, an Astronomy Picture of the Day page displaying a daily space image retrieved via NASA's API, and a Mars Rover Photos page presenting a gallery of images taken by NASA’s rovers. This project also provides a hands-on opportunity to learn about APIs, JSON, and interactive web development.

Scenarios

  1. How can I view today's Astronomy Picture of the Day from NASA?
  2. Where can I browse images captured by NASA's Mars rovers?

Color Schema

  • Primary Color: #003366 – Used for headings, navigation, and borders.
  • Secondary Color: #FFCC00 – Used for accents such as highlights, links, and secondary backgrounds.

These colors are applied throughout this document where appropriate.

Typography

  • Arial: Used for all content including headings and body text, ensuring readability and wide availability.
